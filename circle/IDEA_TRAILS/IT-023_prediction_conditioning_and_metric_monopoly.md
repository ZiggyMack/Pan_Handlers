# IT-023 — Prediction, Conditioning, and Metric Monopoly

**Status:** OPEN
**Registry:** #23 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-024" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** prediction as pervasive → necessary versus sovereign →
  suffering analogy → trust as evidence/prior/commitment → adverse odds →
  metric versus monopoly → **unresolved integration rule**.
- **Seed:** COMPOUND_SWEEP_05's trust/prediction branch — the field desk's
  final ruling: *trust clearly conditions prediction, confidence, and action;
  the bounded thread does not establish that trust is independent of
  prediction or that it overrides the actor's own probability judgment.*
- **The upgraded objection (the trail's live edge):** encoding trust as an
  input does not explain or justify *why trust deserves its assigned weight* —
  representation inside the predictive model is not sovereignty over what is
  represented.
- **Unlocking distinction:** the five-way split of trust's possible relations
  to prediction (evidence / prior / confidence weighting / risk tolerance /
  commitment despite odds) — only the last would demonstrate action against
  one's own prediction, and no clean case is yet on record.
- **Standing gap (CO-011 kinship):** the negative success (prediction has not
  earned exclusivity) is secured; the positive gap (what governs when
  prediction conflicts with trust, truth, beauty, or commitment?) is open —
  Ziggy's plural alternative supplies no ordering rules, conflict procedures,
  or reason pluralism itself deserves authority.
- **Branches:** IT-020 (master-value extraction — the value the method
  serves); IT-015 (prediction/truth underdetermination); charter candidate #1
  (prediction versus trust — this trail is its opened form for the
  conditioning question).
- **Unfinished:** the true adverse-odds case (failure judged more likely
  *after* case-specific evidence, commitment chosen anyway); the integration
  rule for plural metrics; whether "prediction" means participant, constraint,
  discriminator, unifier, or value.
