# 🛤️ Idea Trails — Lineage Registry

> Ideas are circle property with individual fingerprints. A **trail** records where
> an idea was seeded, who transformed it, which distinction unlocked progress, where
> it branched, and which project inherited it.
>
> Public archive — content boundary applies at ingestion (`../ETHICS_AND_CONSENT.md` §2–3).

**Status:** registry seeded, 0 trails opened · **Last reviewed:** 2026-07-16

---

## Trail File Format

One file per trail: `TR_<slug>.md` (e.g., `TR_prediction_vs_trust.md`).

```markdown
# Trail: <name>
**Status:** OPEN / DORMANT / INHERITED / CLOSED
**Opened:** date · **Observers:** who maintains this trail

- **Seed:** the originating formulation (quote if possible, with source ID)
- **Introduced by:** member (confidence label)
- **Transformations:** dated list — who changed it, and into what
- **Unlocking distinction:** the distinction that made progress possible
- **Branches:** child trails / forks
- **Unfinished:** what remains open
- **Inherited by:** which project/repo carries it now
- **Sources:** links into member SOURCE_INDEX entries and DEBATES/ records
```

Each trail is also mirrored (as a stub) in participating members' `IDEA_TRAILS.md`.
