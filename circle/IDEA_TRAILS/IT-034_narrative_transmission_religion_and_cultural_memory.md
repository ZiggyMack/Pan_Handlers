# IT-034 — Narrative Transmission, Religion, and Cultural Memory

**Status:** OPEN
**Registry:** #34 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-035" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** story versus instruction → retention → embodiment →
  identity → distortion → intergenerational transmission.
- **Seed:** COMPOUND_SWEEP_07's transmission branch — the
  Narrative-Transmission Fit Audit ("story vs cookbook"): *same explicit
  lesson ≠ same mnemonic effect ≠ same identity formation ≠ same emotional
  uptake ≠ same cultural persistence.*
- **Unlocking distinction:** the question shifts from *can the teaching be
  restated?* to *can the replacement reproduce the original
  representational functions?* — a legitimate research framework, never a
  superiority proof.
- **Comparative discipline (binding):** manuals optimize explicit
  precision; stories may optimize retention, motivation, identification,
  adaptive reuse — with costs (interpretive drift, ideological capture,
  authority concealment, projection, survival of memorable falsehoods). No
  narrative-superiority claim without **equalized comparison** (ritual,
  childhood exposure, sacred authority, institutional embedding, centuries
  of interpretation) and a **harmful-transmission audit** (A3 omission:
  error correction preserves propaganda, prejudice, and caste as faithfully
  as wisdom).
- **Religion caution:** "religion encodes experiential regularities" = one
  analytical lens among ritual, norms, identity, cosmology, authority,
  coordination — the claim-strength ladder applies (possible ≠ important ≠
  principal).
- **Branches:** IT-026 (compression ladder); IT-037 (transmission fitness
  vs content quality — this trail's evaluative face); AUX_05's story/
  orientation material; charter candidate #4.
- **Unfinished:** the equalized story-vs-manual comparison under matched
  conditions; separating religious persistence from institutional power;
  whether secular narratives (the Star Wars case) can carry the same
  functional load.
