# IT-091 — Moderation Before Emergent Value Is Known

**Status:** OPEN
**Registry:** #91 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-094" — numbering drift; registry id is canonical.)*

- **Core trajectory:** uncertain future value → present cost → bounded
  experimentation → boundary response → continue or terminate.
- **Seed:** SWEEP_12's moderation problem — the unanimity standard
  rejected (*any participant would gain veto power; others absorb
  unlimited costs*); moderation operates prospectively under
  uncertainty.
- **The mature principle (binding):** *do not suppress inquiry merely
  because its value is not yet visible — but do not require participants
  to endure unlimited disruption because valuable ideas might eventually
  be salvaged.*
- **The weighing factors carried:** expected informational value ·
  current harm · cost distribution · participant consent · lower-cost
  alternatives · responsiveness to boundaries · reversibility · time
  limits · moderator responsibility.
- **The salvageability clause (confirmed A3-11):** *recoverable value ≠
  continued-access right* — has the useful material already been
  extracted? would boundaries preserve the gain at lower cost?
- **The closure split (confirmed A3-12):** premature epistemic closure ≠
  prudent moderation ≠ participant boundary ≠ fatigue ≠ disinterest —
  the discipline that keeps the suppression narrative honest.

  IT-092 (where the suppression reading goes when unguarded).
