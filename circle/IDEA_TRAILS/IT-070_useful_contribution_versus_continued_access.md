# IT-070 — Useful Contribution versus Continued Access

**Status:** OPEN
**Registry:** #70 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-071" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** disruptive input → informational value extracted →
  costs assessed → behavior evaluated → contribution retained →
  participant redirected or removed.
- **Seed:** SWEEP_10's decisive principle — *a contribution's
  informational value and its source's continued participation are
  separate decisions.* The conversation may keep the useful
  counterexample while excluding the person who continually derails.
- **The five outcomes carried:** retain / redirect / **useful
  contribution + remove participant** / warn / terminate.
- **Both reciprocals permanent:** *useful noise ≠ right to remain* AND
  *removal ≠ the contribution had zero information value.*
- **Branches:** IT-063 (the policy audit this decision implements);
  IT-071 (the thresholds); ETHICS_AND_CONSENT §2–3 (the archive's own
  version: content boundary at ingestion — extract the idea, guard the
  person).
- **Unfinished:** who makes the access decision; appeal; whether the
  extraction-then-exclusion move is itself fair to the excluded (the
  moral remainder Angles' objection gestures at).
