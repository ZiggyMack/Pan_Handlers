# IT-037 — Transmission Fitness versus Content Quality

**Status:** OPEN
**Registry:** #37 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-038" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** memorability → redundancy → cultural persistence →
  institutional reinforcement → truth → wisdom → moral quality.
- **Central question:** *what does successful cultural survival actually
  provide evidence for?*
- **Seed:** COMPOUND_SWEEP_07's deepest theory-level correction —
  *transmission success ≠ truth tracking ≠ wisdom preservation ≠ moral
  legitimacy.* An error-correcting code preserves a message against noise;
  it does not determine whether the preserved message is good.
- **Unlocking distinction — two separate evaluations, never collapsed:**
  1. **Transmission quality** — how faithfully and durably does the pattern
     propagate?
  2. **Content quality** — is the propagated pattern true, adaptive,
     humane, or wise?
  The original conversation moved too quickly from the first to the second
  (the sweep's strongest confirmed A3 omission: fitness-vs-truth never
  applied to mythic survival).
- **The negative-case ledger (binding):** cultural error correction may
  preserve wisdom, dogma, identity, prejudice, ritual, power, propaganda,
  caste, scapegoating, authoritarian loyalty, or falsehood — with equal
  fidelity. Survival explanations to be separated: memorability, ritual
  reinforcement, institutional power, childhood exposure, coercion,
  political authority, literary excellence, selection effects, practical
  wisdom.
- **Branches:** IT-034 (the transmission mechanisms this trail evaluates);
  IT-026 (falsifiability debt — the same discipline for compression);
  IT-017 (moral generativity — content quality's hardest case).
- **Unfinished:** any metric separating fitness from quality in real
  corpora; the deepest yield's challenge — *"a pattern exceptionally fit to
  reproduce"* as the null hypothesis for every surviving tradition.
