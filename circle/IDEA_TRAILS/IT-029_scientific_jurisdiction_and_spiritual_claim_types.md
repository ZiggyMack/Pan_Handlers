# IT-029 — Scientific Jurisdiction and Spiritual Claim Types

**Status:** OPEN
**Registry:** #29 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-030" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** scientific success → higher-order adjudication →
  is/ought boundary → first-person evidence → metaphysical interpretation →
  empirical consequences.
- **Seed:** COMPOUND_SWEEP_06's jurisdiction branch — "is the atheist taking
  the theistic idea into its frame to die?" — with the field desk's
  two-sided test: hostile translation (a genuine category error IF the claim
  never posited a detectable object) vs legitimate consequence testing
  (claims with empirical consequences cannot evade scrutiny by being housed
  in a spiritual frame).
- **Unlocking distinction:** the **mixed-claim table** — "spirituality" is
  not one epistemic jurisdiction: phenomenological / psychological /
  historical / causal / metaphysical / normative / theological claims take
  different primary evidence; *no single method governs all rows; no row is
  exempt because it appears inside a religious narrative.*
- **The balance point (permanent archive ruling):** *scientific authority
  without scientific sovereignty; spiritual evidence without spiritual
  immunity.* Whatever super-frame scrutiny is applied to science must be
  applied symmetrically to testimony, qualia, and revelation — the packet's
  strongest confirmed omission (A3) was withholding that audit from the
  spiritual side.
- **Method rule:** reconstruct the claim in its own ontology → identify what
  it actually predicts or entails → apply standards appropriate to those
  claimed consequences — prevents both scientism and framework immunity.
  Framework-Translation Fidelity Check stays provisional (suspicion
  present; completed distortion demonstration absent).

  IT-031 (experience → public authority); SW03's jurisdiction vein (the
  justificatory face).
- **Unfinished:** the decisive Stage-4 question — *what principled rule says

  incompatible spiritual testimonies are adjudicated; sub-dig C = strongest
