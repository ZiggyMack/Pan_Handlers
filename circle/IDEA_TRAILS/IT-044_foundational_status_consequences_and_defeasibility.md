# IT-044 — Foundational Status, Consequences, and Defeasibility

**Status:** OPEN
**Registry:** #44 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-045" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** basic commitment → derived consequences →
  evidential pressure → revision of auxiliary claim → revision of
  foundation → **possible insulation**.
- **Seed:** DIG_AUX_07's Consequence–Status Decoupling in both directions:
  a foundational proposition may generate predictions without ceasing to
  be foundational, AND foundational status does not immunize the
  proposition from indirect evidential pressure.
- **The six-step religious test case (registered, never run):** identify
  the alleged axiom → state its foundation type → derive one specific
  consequence → classify it (empirical / moral / existential / historical
  / metaphysical) → state what evidence bears on it → explain how failure
  propagates to the derived claim vs the foundation.
- **Worked examples carried:** *providence → specific prayer-recovery
  rates* (failure pressures the specific causal doctrine, not every
  religious proposition) vs *pride corrupts → broad psychological
  expectations* (not a precise physical forecast) — different rows,
  different jurisdictions.
- **The insulation watch (both failure modes):** "axiomatic" as an
  evidence-shield (the dogma problem — what distinguishes a properly
  basic belief from a protected assertion?) vs consequence-testing as a
  category bulldozer (the reported inference *has consequences → not
  axiomatic* remains bridgeless).
- **Branches:** IT-039 (foundational status); IT-029 (mixed-claim table —
  the classification step's instrument); IT-024 (operative authority —
  what enforcement does to revisability); IT-017 (moral generativity —
  the pilot's hardest derived-claim case).
- **Unfinished:** the test case actually run on a named religious
  proposition; the propagation rules (when does derived failure reach the
  foundation?); whether any surviving foundation is revisable in
  practice, not just in principle.
