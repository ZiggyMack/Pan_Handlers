- **Core trajectory:** local perspective → relational global structure →
  belief facts → world facts → epistemic access.
- **Seed:** COMPOUND_SWEEP_06's observer branch — the **teapot
  decomposition**, ruled the cleanest part of the packet and an important
  **anti-relativist correction**: *"A believes X" and "B believes not-X" can
  both be true; X and not-X cannot both describe the same causal state in
  the same respect.*
- **Unlocking distinction:** mental-state facts / world-state facts /
  epistemic facts — truth about a representation ≠ truth of what it
  represents (Representation ≠ Ontology; Locate Disagreement Layer
  adjacency).
- **The global-structure guard:** *many perspectives in one world ≠ many
  worlds*; global structure = relations, transformations, and constraints
  among perspectives — NOT a supreme observer, NOT a multiverse, and NOT a
  complete truth theory (survives only for perspective-indexed differences).
  Nova's Relational Globalization stays an **Idea-Trail proposal**, never a
  Museum operator, until a formal model appears.

  supernova-to-teapot transition (permissive case → discriminating case →
  scope boundary); provisional until bounded recurrence.
