# IT-093 — Does Universal Uncertainty Trivialize Faith?

**Status:** OPEN (Idea Trail)
**Registry:** #93 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-096" — numbering drift; registry id is canonical.)*

- **Core question:** if every action containing residual uncertainty
  counts as faith, what does *not* count as faith? A concept that
  excludes nothing discriminates nothing.
- **Seed:** SWEEP_13's central dispute — Ziggy's "atheists have faith
  too" argument (faith = acting under uncertainty; all action is
  uncertain) vs Grant's overbreadth challenge (the son-height
  prediction, the spoiled-milk case).
- **The trap (binding, confirmed A3-4):** *all action contains
  uncertainty* + *faith means action under uncertainty* → *all action
  is faith* — possibly a substantive discovery, possibly a conclusion
  **true by stipulation rather than discovery**. The definition decides
  the result before evidence is consulted.
- **Unlocking distinction:** the narrowing conditions (action guidance,
  practical reliance, stakes, unavoidable choice, identity-level
  commitment) rescue discriminating content — but their *timing* is the
  live issue: original criteria, or post-counterexample additions?
  Status `UNKNOWN` (the principled-vs-ad-hoc question the bounded
  source does not decide).
- **Kinship:** IT-094 (threshold separation); IT-100 (what completes
  evidence into action); the reflexive-falsifiability instrument (a
  definition immune to counterexample is interpretive capture).
- **Unfinished:** a frozen necessary-condition set for "faith" stated
  *before* example scoring; whether functional reclassification ever
  earns the word, or only borrows its connotations.
