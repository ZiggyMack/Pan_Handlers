# IT-112 — Retained Justification States and the Forgotten Derivation

**Status:** OPEN (Idea Trail)
**Registry:** #112 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)

- **Core category (SW15's freshest yield):** **retained higher-order
  evidence from one's past reasoning** — the differential-equation
  case: a person remembers *having verified* a conclusion through
  competent reasoning but cannot presently reproduce the derivation.
- **The five-way separation (binding):** past justification ≠ present
  first-order memory of the argument ≠ higher-order memory ("I
  previously verified this") ≠ current confidence ≠ current public
  demonstrability. Two-sided discipline: *retrieval failure ≠ lying*
  — AND — *"I once proved it" ≠ current public proof; retained
  confidence ≠ indefeasible warrant* (past-self testimony weakens
  with time, error risk, and stakes).
- **The owed discriminations (A1):** propositional memory /
  procedural knowledge / tacit skill / higher-order verification
  memory / present confidence / current access to reasons /
  reflective equilibrium — the assistant's first labels (tacit
  knowledge, embodied understanding) were only partial fits.
- **The debate-timing norm attached:** a legitimate evidential burden
  need not be satisfiable *instantly* (Retrieval-Time Resistance) —
  but with a boundary owed between delayed obligation and indefinite
  insulation: did the reconstruction ever arrive? did confidence
  scale down while it was pending?
- **Kinship:** IT-048 (belief rescue — the discriminator applies when
  replacement reasons are sought); IT-094 (thresholds); the
  Memory-State Qualification queue row (Ziggy's direct seed, counter
  to Unmeasured-to-Zero Conversion).
- **Unfinished:** confidence-scaling rules; the stopping condition
  for "I'll reconstruct it later"; when higher-order memory alone
  licenses assertion.
