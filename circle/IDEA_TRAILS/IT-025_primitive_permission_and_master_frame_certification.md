# IT-025 — Primitive Permission and Master-Frame Certification

**Status:** OPEN
**Registry:** #25 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-026" — recurring off-by-one; registry id is canonical.)*

- **Core trajectory:** definition demand → primitive exclusion suspicion →
  equal admission → explanatory-cost comparison → local coherence → **global
  certification problem**.
- **Seed:** COMPOUND_SWEEP_05's opening move — "this is presup dressed up as
  method": a demand for non-circular definition may conceal a prior rule
  about which entities are allowed to count as explanatory primitives.
- **Unlocking distinction (two-stage, both directions guarded):**
  *permission to compete ≠ equal explanatory cost ≠ equal evidence ≠ equal
  adequacy* — fair admission to comparison never settles the comparison; AND
  difficulty reducing consciousness ≠ evidence that consciousness is
  fundamental. A clarification request is not automatically an illicit
  reduction demand.
- **The certification problem (the trail's summit):** local coherence or
  utility within a frame never certifies that frame as the unique global
  metaphysics — but "several coherent frames exist" does not establish "no
  global truth exists" either. The missing piece is the gluing/transfer
  principle: what would legitimately promote a locally successful frame to
  global authority?
- **Registered rejection (polarity discipline):** the Primitive-Permission
  Audit does NOT extend Generative Admission Restriction — detecting a
  restriction is not exhibiting one; adjacency runs to Symmetry Testing and
  to Baseline-Exemption *detection*.
- **Branches:** IT-020 (master values — what the certified frame would
  serve); IT-014 (invariance/convergence — what survives across frames);
  ≠-family (Necessary ≠ Supreme candidate; Representation ≠ Ontology).
- **Unfinished:** the reverse-role test never run (beauty, pluralism, or
  trust auditioned as sovereign master — downgraded to A1); whether
  anti-monopoly itself files the same paperwork it demands (Stage 4's own
  symmetry note); the adjudication rule among rival certified frames.
