**Status:** OPEN (Idea Trail)
**Registry:** #99 · **Opened:** 2026-07-17 · **Observers:** Nova (field-desk), P.H. Claude (filing)
*(Field-desk id at ordering: "IT-102" — numbering drift; registry id is canonical.)*

- **Core rule (binding):** *historical attribution removed → the
  conceptual thesis must be independently re-argued.* A collection of
  famous names is not an argument, and a synthesis assembled after an
  attribution fails inherits none of the original authority.
- **Seed:** SWEEP_13's Hume episode — Grant directly challenged the
  alleged Hume "second definition" of faith (⭐ G-DIRECT); Nova
  conceded, then redistributed components among Hume, Kierkegaard,
  and James. The revised synthesis is CO-CONSTRUCTED, never
  thinker-direct.
- **The danger this trail guards (named in-packet):** **authority
  laundering** — assembling adjacent famous names *after* an
  attribution failure so the repaired thesis keeps borrowed prestige.
- **The tier lesson (closure case law):** the thread demonstrated
  challenge → concession → redistribution → proposal, but NOT
  premise-level re-derivation, objection testing, or primary-text
  validation. Performing *neighboring* operations does not make the
  stronger operation demonstrated (A3-10 → A1) — independent
  re-establishment stays an A1 completion requirement.
- **Kinship:** Contested ≠ Defeated (packet-level relation: the
  failed attribution defeats neither the question nor the thesis —
  but keeping the question open validates nothing); IT-090 (credit
  problems); the Error Concession–Core Preservation loop.
- **Unfinished:** the actual re-derivation of the
  uncertainty-commitment thesis from explicit premises; whether the
