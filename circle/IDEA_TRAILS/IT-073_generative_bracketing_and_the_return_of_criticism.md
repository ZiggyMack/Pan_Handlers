- **Core trajectory:** framework assumed → implications developed → debts
  recorded → criticism restored → comparative judgment.
- **Seed:** SWEEP_11's "assume mind is fundamental" sandbox — legitimate
  under four conditions: defined scope · reciprocity (rival frameworks get
  equal sandbox time) · an actual return of criticism · **no hidden
  victory condition** (sandbox coherence ≠ evidence of truth).
- **Failure mode:** **Bracketing-to-Privilege Drift** (rich framework →
  psychological entrenchment → criticism stays "premature" → temporary
  bracket becomes ontological privilege). A complete sandbox rule needs
  start, scope, duration, exit, comparison standard, reciprocity.

  specified (A3-5), and sandbox reciprocity was never tested (A3-6) —
  *exploratory intent supported; implementation fairness unestablished.*
