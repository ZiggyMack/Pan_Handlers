## Norm origins registered so far

- **Contested ≠ Defeated** — imported from the Nyquist Museum ledger discipline; institutionalized as the `CONTESTED` label (charter, 2026-07-16).
- **Old chats = ruins; fresh threads = museum and integration chamber** — Nova, 2026-07-16 ("sending archaeological crews back into the original ruins").
- **Nova's language is attributable to Nova** — the NOVA-INTERPRETATION label exists so assistant formulations are never silently credited to members (Nova, self-imposed, 2026-07-16).
- **"This is an execution command, not a request for protocol review"** — the mode-control wrapper preceding every dig stage, born from the v0.1 field test where the extractor polished the archaeological brush while the ruins sat untouched (Nova, 2026-07-16).
- **Blind before named** — extraction runs Museum-blind (Stage 2) before candidate names appear (Stage 4); a blind rediscovery outranks a primed match (Nova, 2026-07-16).
- **Evidence before instrument revision** — once an instrument is field-worthy, further refinement before first contact with evidence is its own avoidance operator; freeze design, run the dig (Nova, 2026-07-17).
- **Contamination is provenance, not noise** — leakage gets an honest reporting channel (X-CONTEXT) and becomes prospecting; enforcement lives in anchor verification, never in the extractor's self-report (P.H. Claude, ratified by Nova, 2026-07-17).
- **Titles lie — preflight before digging** — reused chats drift from their names; a human skim of first/last messages precedes every Stage 1. A wrong thread is a preflight failure, not a protocol failure, and a scope-clean wrong-target dig is repurposed as an AUX dig, never discarded (Nova, 2026-07-17).
- **Stage 1 is the sorting hat** — every bounded Stage 1 classifies its chat: Grant trench / auxiliary dig / idea-trail seed / orientation sweep / non-site. Classification is a valid outcome even when the intended target is absent (Nova, 2026-07-17).
- **Excavate generously, classify conservatively** — Stage 2 may name thirteen moves; Stage 4 registers two. Phases, outputs, and rhetoric are not operators (Nova, 2026-07-17).
- **Adjacent in spirit is not lineage** — family resemblance to an existing operator is not an EXTENDS verdict; a mapping needs the defining conditions (for `Contested ≠ Defeated`: an actual contest), not a vibe match (Nova, 2026-07-17).
